Constructor

Constructor adalah method khusus dalam sebuah class yang digunakan untuk menginisialisasi object.
Constructor akan otomatis dipanggil saat sebuah object dibuat.

Ciri-ciri Constructor
1. Nama constructor harus sama dengan nama class
2. Tidak memiliki return tipe (bahkan void pun tidak boleh ditulis)
3. Bisa memiliki parameter untuk menerima nilai saat object dibuat

Bukti bahwa constructor pertama kali panggil:
class Mobil {

    String merek;
    String warna;

    Mobil() {
        System.out.println("constructor called");
    }

}

class Main {

     public static void main(String[] args) throws Exception {

        new Mobil(); // output: constructor called

    }
}

Contoh membuat constructor:
class Main {
    public static void main(String[] args) {
        Mobil mobil1 = new Mobil("Toyota", "Merah);
        System.out.println(mobil1.displayMobil());
    }
}

class Mobil {
    String merek;
    String warna;

    Mobil(String merek, String warna) {
        this.merek = merek;
        this.warna = warna;
    }

    String displayMobil() {
        return "Mobil merek : " + this.merek + " Warna : " + this.warna;
    }
}


Kelebihan menggunakan constructor:
Constructor dibuat untuk menginisialisasi object secara otomatis saat object dibuat. Tanpa constructor
kita harus mengatur nilai atribut satu per satu setelah membuat object (merepotkan dan rawan kesalahan).

Contoh tidak menggunakan constructor
Mobil mobil1 = new Mobil();
mobil1.merek = "Toyota";  // Harus diisi manual
mobil1.warna = "Hitam";

Dengan constructor
Mobil mobil1 = new Mobil("Toyota", "Hitam"); // tanpa mengisi attribut satu per satu

Contoh tanpa constructor
Mobil mobil1 = new Mobil();
mobil1.merek = "Toyota";
mobil1.warna = "Hitam";

Mobil mobil2 = new Mobil();
mobil2.merek = "Honda";
mobil2.warna = "Merah";

Mobil mobil3 = new Mobil();
mobil3.merek = "Nissan";
mobil3.warna = "Putih";


Contoh menggunakan constructor
Mobil mobil1 = new Mobil("Toyota", "Hitam");
Mobil mobil2 = new Mobil("Honda", "Merah");
Mobil mobil3 = new Mobil("Nissan", "Putih");
