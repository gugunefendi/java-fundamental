Method
Method dengan keyword static berarti methodnya dapat di panggil nempel dengan classnya.
Kalau method ingin buat method yang nempel dengan objectnya jangan gunakan static 
tapi gunakan void

Method ada 4 tipe
1. Method tanpa return dan tanpa paramater
2. Method tanpa return dan dengan paramater
3. Method dengan return tapi tidak ada parameter
4. Method dengan return dan dengan parameter


Contoh:
class Mobil {
    String merek;
    String warna;

    Mobil(String merek, String warna) {
        this.merek = merek;
        this.warna = warna;
    }

    // method tanpa return dan tanpa parameter
    void showMobil() {
        System.out.println("Merek mobil : " + this.merek);
    }

    // method tanpa return dan dengan parameter
    void setMerek(String merek) {
        this.merek = merek;
    }

    void setWarna(String warna) {
        this.warna = warna;
    }

    // method dengan return dan tanpa parameter
    String getMerek() {
        return this.merek;
    }

    String getWarna() {
        return this.warna;
    }

    // method dengan return dan dengan parameter
    String displayMobil(message) {
        return message "dengan merek : " + this.merek + "warnanya adalah : " + this.warna;
    }
}
